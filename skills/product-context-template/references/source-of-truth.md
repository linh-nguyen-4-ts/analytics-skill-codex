# Source of Truth

Record which tables or reports should win when numbers disagree.

## Recommended Format

| Topic | Source of truth | Why it wins | Known exceptions | Owner |
|-------|-----------------|-------------|------------------|-------|
| Example: application volume | `dop.applications` | authoritative creation record | excludes archived test rows | Data team |

## Questions to Answer

- Which table is authoritative for each KPI?
- Which dashboard or report is considered official?
- When two tables disagree, what explanation is expected?
- Which team owns the final definition?
